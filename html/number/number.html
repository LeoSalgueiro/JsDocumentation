Number

Number es un objeto primitivo envolvente que permite representar y manipular valores numéricos cómo 37 o -9.25. El constructor Number contiene constantes y métodos para trabajar con números. Valores de otro tipo pueden ser convertidos a números usando la función Number().

            new Number(value);
            var a = new Number('123'); // a === 123 es false
            var b = Number('123'); // b === 123 es true
            a instanceof Number; // es true
            b instanceof Number; // es false

ParseInt

El método Number.parseInt() analiza un argumento de tipo String y devuelve un número entero de la raíz o base especificada.

×

Number.parseInt()


Sintaxis

Number.parseInt(string,[ radix])

Parámetros


El valor a analizar. Si este argumento no es una cadena, entonces se convierte en uno usando la ToString operación abstracta. Se ignoran los espacios en blanco iniciales en este argumento.


Un número entero entre 2 y 36 que representa la raíz (la base en los sistemas numéricos matemáticos) del string. Tenga cuidado, ¡esto no se establece de forma predeterminada en 10!


Valor de retorno

Un entero analizado del dado string.


Si radix es menor a 2 o mayor que 36 y el primer carácter que no es un espacio en blanco no se puede convertir en un número, entonces se devuelve NaN

Division Remainder (%)

Los arrays son objetos similares a una lista cuyo prototipo proporciona métodos para efectuar operaciones de recorrido y de mutación. Tanto la longitud como el tipo de los elementos de un array son variables. Dado que la longitud de un array puede cambiar en cualquier momento, y los datos se pueden almacenar en ubicaciones no contiguas, no hay garantía de que los arrays de JavaScript sean densos; esto depende de cómo el programador elija usarlos.
